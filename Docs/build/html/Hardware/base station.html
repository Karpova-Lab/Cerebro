Uploading Firmware¶

  1. With the notch facing down, place the chip in the lower portion of the Utility Shield‘s ZIF socket.
  1. Open the Arduino Software and select Tools -> Board: -> Base Station (ATtiny85)
  1. Choose the COM port that your Arduino Uno is connected to by selecting Tools -> Port: ->
  2. Select Tools -> Programmer -> Arduino as ISP
  3. Burn the bootloader by selecting Tools -> Burn Bootloader
  1. Choose the Base Station firmware by going to File -> Sketchbook -> Cerebro Sketches -> baseStation
  1. Upload the firmware by clicking the upload arrow
